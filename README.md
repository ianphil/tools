A collection of single-file browser-based tools.

The [colophon](colophon) lists commit messages and transcripts for every tool.

## Text Tools

- [Word Counter](word-counter) - Count words, characters, and lines

## Network Tools

- [Subnet Calculator](subnet-calc) - Calculate subnet info using CIDR ranges

## Azure Tools

- [ARM/Bicep Resource Reference](arm-bicep-reference) - Quick lookup for Azure resource types and API versions

## Encryption Tools

- [Base64 Encoder/Decoder](base64-tool) - Tool for encoding or decoding base64
- [Sha256 Hash Tool](sha256-hash-tool) - Tool for hashing text or files using sha256
